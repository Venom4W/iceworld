a fast restful dev framework base on tornado

Take advantage of a fashionable C10K framework Turnado(ioloop). ioloop throws requests to threadpool, threadpool notifies ioloop when it is done.

1.In "biz" folder, make a py file from demo which existed there

2.Use "Router.route" decorate a method of your business class after import Router module

3.Goto "bin" folder, the use command --"python serv.py" to start up

enjoy!
